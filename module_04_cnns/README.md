# Module 4: Convolutional Neural Networks (CNNs)

## Overview

CNNs are the foundation of computer vision. This module covers CNN architecture, convolution operations, and practical applications in image classification.

## Learning Objectives

- Understand convolution operations
- Master CNN architecture components
- Implement CNNs from scratch
- Build image classifiers with PyTorch/TensorFlow
- Apply transfer learning
- Visualize learned features
